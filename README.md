## 🎨 Design Philosophy: "Technical Empathy"
In my work, I follow three strict rules:
1.  **Accessibility First:** Clean contrast and readable typography (Inter) are non-negotiable.
2.  **No-Fluff UI:** If a feature doesn't solve a user problem, it’s a distraction.
3.  **Humanized Code:** Documentation and interfaces should speak a language everyone understands.
